📄 11個頁面的完整詳細說明

1️⃣ 首頁 (HOME)
URL: /
🔗 可點擊元素
LOGO (皇冠) → 留在首頁
簽證常見問題▼ → 部落格頁 (Blog)
關於簽證王 → About頁
登入 → Login頁
商品卡「立即申辦」 → 商品頁 (Products)
CTA「立即申辦」 → 商品頁 (Products)
CTA「立即諮詢」 → LINE官方帳號 外連
2️⃣ 部落格頁 (BLOG)
URL: /blog/[article-slug]
🔗 可點擊元素
推薦簽證卡 → 商品頁 (Products)
上一頁 → 首頁 返回
3️⃣ 關於簽證王頁 (ABOUT)
URL: /about
🔗 可點擊元素
Banner「立即申辦」 → 商品頁 (Products)
Banner「立即諮詢」 → LINE官方帳號 外連
Google Map → Google Map 外連
CTA「立即申辦」 → 商品頁 (Products)
LOGO → 首頁
4️⃣ 登入/註冊頁 (LOGIN)
URL: /login
🔗 認證流程
Google OAuth 登入 → 簽證選擇頁 自動跳轉
帳號未註冊 → 自動註冊並登入
💡 特殊說明： 登入完成後自動導回原本路徑。如是從簽證選擇頁跳轉，登入後會自動返回簽證選擇頁（無需重新選擇簽證）
5️⃣ 商品頁 (PRODUCTS)
URL: /products
🔗 可點擊元素
商品卡「立即申辦」 → 簽證選擇頁
LOGO → 首頁
返回 → 首頁 返回
6️⃣ 簽證選擇頁 (VISA SELECTION)
URL: /country/[country-name]
📋 頁面內容
出發日期選擇 → 更新可選簽證類型
簽證類型選項 → 選擇後啟用下一步
🔗 導航按鈕
下一步 → 類型確認頁 需選擇日期和類型
上一步 → 商品頁 保留資料
7️⃣ 簽證類型確認頁 (TYPE CONFIRMATION)
URL: /confirm
📋 自動帶出欄位（全為唯讀）
簽證類型 → 唯讀，不可修改
出發日期 → 唯讀（訂單確認頁才能更新）
工作天數/費用 → 自動判定（1日/2日/急件），唯讀
🔗 導航按鈕
下一步 → 訂單確認頁
上一步 → 簽證選擇頁 保留資料
💡 說明： 此頁面所有欄位為唯讀。如需更新出發日期，請進入訂單確認頁進行修改。
8️⃣ 訂單確認頁 (ORDER CONFIRMATION)
URL: /order
📋 必填區塊
[出發日期更新] → 日曆模式更新日期 自動重新計算費用
[申請人資料] → 名字、電話(09開頭)、Email(@gmail/@yahoo)
+新增申請人 → 新增多位申請人 單一訂單
[LINE ID] → 所有申請人共用一組
[取件方式] → 自取/普掛(免)/限掛(¥100)/快捷(¥150)
[付款方式] → 信用卡/Apple Pay/Google Pay/LINE Pay
🔗 導航按鈕
下一步 → 付款頁 驗證通過
上一步 → 類型確認頁 保留資料
💡 日期更新說明： 用戶可在此頁面點擊日期日曆更新出發日期。更新後系統會自動重新計算工作天數和簽證費用，費用變更會立即在付款摘要中反映。
9️⃣ 付款頁 (PAYMENT)
URL: /payment
📋 頁面內容
[訂單明細] → 自動帶出（簽證費、工作天數等）
[付款方式] → 信用卡/Apple Pay/Google Pay/LINE Pay
[結帳金額] → 簽證費 + 運費 = 總金額
🔗 導航按鈕
確認付款 → 第三方支付閘道 → 訂單查詢頁 自動跳轉
上一步 → 訂單確認頁 保留資料
🔟 訂單查詢頁 (ORDER QUERY)
URL: /orders
📋 頁面內容
訂單列表 → 顯示所有訂單
點擊訂單 → 查看訂單詳情、下載文件、追蹤進度
🔗 導航按鈕
LOGO → 首頁
上一步 → ❌ 無此按鈕 防誤操作
1️⃣1️⃣ 錯誤頁 (ERROR PAGE)
URL: /error (或無效URL)
📋 頁面內容
錯誤訊息 → 「404頁面未找到」或「發生錯誤」
倒數計時 → 顯示「3秒後自動返回...」 可中斷
🔗 用戶操作選項
點擊「返回上一頁」按鈕 → 回到訪問此錯誤頁前的頁面
點擊「LOGO」 → 首頁
⏱️ 自動操作
3秒倒數完成 → 自動返回上一頁（用戶可點擊任何按鈕中斷）
💡 流程說明： 錯誤頁面會自動在3秒後將用戶導回「訪問錯誤頁前的上一個頁面」。用戶可在倒數過程中點擊「返回上一頁」按鈕立即返回，或點擊「LOGO」直接回到首頁。

🔑 重要流程說明：

✓ 登入檢查機制： 用戶在簽證選擇頁選擇簽證類型後，系統會檢查登入狀態。如未登入→導向Login頁→登入完成後自動返回簽證選擇頁（無需重新選擇）

✓ 日期自動判定： 簽證類型確認頁會根據出發日期自動判定是否可快遞（1日）或常規（2日），並自動更新費用和工作天數。用戶進入訂單確認頁後，仍可點擊日期重新調整，系統會自動重新計算費用

✓ 日期更新流程： 在訂單確認頁頂部的「出發日期更新」區塊可修改日期。更新後費用會自動重新計算，用戶可在付款前看到最終費用

✓ 數據保留機制： 除了「簽證選擇頁返回商品頁會清空」外，所有返回上一步都會保留已填寫的資料

✓ 多申請人訂單： 用戶可在同一訂單中申請多位簽證，但所有申請人共用一組LINE ID

✓ 訂單查詢頁特殊性： 此頁面無「上一步」按鈕，防止用戶誤操作返回修改已完成的訂單
